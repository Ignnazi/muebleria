# E2-Muebleria
”Mueblería los muebles hermanos S.A” es un negocio que comercializa distintos tipos de muebles (sillas, sillones, mesas, estantes y cajones), con diferentes variaciones (material, tamaño y estilos). Gustavo quien es gerente necesita un Back-End que permita registrar, cotizar y vender estos muebles de forma consistente.
